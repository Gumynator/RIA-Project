Description du projet

Par Loan Rey, Massimo Pantucci, Pedro Fonseca le 07/04/2022

Description

Notre jeu est basé sur le jeu classique “Snake”. L’espace dédié au jeu est délimité par des parois. Le joueur dirige le vecteur qui s’agrandit au fur et à mesure qu’il s’alimente des éléments présents dans la carte.

But du jeu

Le but du jeu est de créer le plus grand vecteur possible en évitant les bordures et lui-même.

Personnages

Le personnage est en réalité un objet qui s’agrandit.

Règles du jeu

L’objet ne peut pas toucher les bordures ainsi que lui-même.

Le code source de l'application est disponible sur le github du projet.
